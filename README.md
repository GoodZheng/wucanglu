# 物藏录 (wucanglu)

> 个人物品与订阅管理工具 — uTools 插件

![Version](https://img.shields.io/badge/version-2.0.0-blue)
![Platform](https://img.shields.io/badge/platform-uTools-green)

## 简介

物藏录是一款简洁高效的个人物品与订阅管理工具，以 uTools 插件的形式运行，帮助你轻松管理生活中的物品和订阅服务，追踪消费支出。

### 主要功能

#### 📦 物品管理
- **添加物品**：记录物品名称、价格、分类、购买渠道等信息
- **分类管理**：自定义物品分类，便于组织和查找
- **物品统计**：可视化图表展示物品分布和消费情况

#### 💳 订阅管理
- **订阅追踪**：管理各类订阅服务（视频、音乐、软件等）
- **扣费日历**：直观展示每月扣费日期和金额
- **周期支持**：支持按天/周/月/季度/年/终身等多种计费周期
- **智能提醒**：到期前自动提醒，避免忘记续费或取消
- **消费分析**：统计订阅支出，掌控每月/每年的订阅开销

#### 📊 数据分析
- **物品分析**：按分类统计物品数量和价值
- **订阅分析**：按周期统计订阅支出分布
- **图表展示**：直观的饼图和柱状图

#### ⚙️ 设置
- **数据导出/导入**：支持 JSON 格式的数据备份和恢复
- **数据清空**：一键清空所有数据

## 安装

### 方式一：uTools 插件市场（推荐）
1. 打开 [uTools](https://u.tools/)
2. 在插件市场搜索「物藏录」
3. 点击安装即可使用

### 方式二：本地开发
1. 克隆本仓库到本地
2. 打开 uTools，进入「插件开发」模式
3. 加载插件目录即可

## 使用说明

### 物品管理
1. 点击右下角 `+` 按钮添加物品
2. 填写物品信息（名称、价格、分类、渠道等）
3. 支持编辑和删除物品
4. 点击「管理分类」可自定义物品分类

### 订阅管理
1. 添加订阅：填写名称、价格、计费周期、首次扣费日期
2. 扣费日历：点击任意日期查看当天扣费详情
3. 编辑订阅：在详情列表中点击「编辑」按钮
4. 管理分类：自定义订阅分类

### 订阅周期说明
| 周期类型 | 说明 | 示例 |
|---------|------|------|
| 按天 | 每天扣费 | 会员卡按天计费 |
| 按周 | 每周扣费 | 每周杂志订阅 |
| 按月 | 每月扣费 | 视频会员月卡 |
| 按季度 | 每季度扣费 | 软件订阅季卡 |
| 按年 | 每年扣费 | 会员年费 |
| 终身 | 一次性付费 | 永久软件授权 |

## 技术栈

- **前端框架**：原生 JavaScript (Vanilla JS)
- **样式**：原生 CSS
- **图表库**：[Chart.js](https://www.chartjs.org/)
- **数据存储**：uTools DB / localStorage（本地存储）
- **构建工具**：无需构建，单文件部署

## 开发说明

### 本地开发
1. 直接编辑 `index.html` 文件
2. 在 uTools 开发模式下重新加载插件查看效果
3. 无需构建过程，即改即生效

### 代码规范
- 使用类模块化架构（App、Storage、SubscriptionUI 等）
- 遵循 [CLAUDE.md](./CLAUDE.md) 中的开发约定
- 日期处理使用 `YYYY-MM-DD` 格式，避免 UTC 时区问题
